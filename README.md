Alpha Risk Solutions è un Software web full-stack API-based per la gestione e la preventivazione assicurativa.

Descrizione del progetto:
Il progetto è stato sviluppato come applicazione web full-stack API-based per supportare la digitalizzazione dei processi gestionali ed economico-finanziari in un’organizzazione del settore sanitario.

Tecnologie utilizzate:
- ASP.NET Core (.NET 8)
- C#
- SQL Server
- Entity Framework Core
- Identity
- Bootstrap
- Swagger/OpenAPI

Le Funzionalità principali:
- autenticazione e gestione ruoli
- gestione macrocategorie
- gestione polizze e servizi aggiuntivi
- gestione documenti
- creazione e consultazione preventivi
- documentazione API tramite Swagger

La Struttura del progetto:
- Preventivatore.Api
- Preventivatore.Core
- Preventivatore.Infrastructure
- Preventivatore.WebApp

Note:
Il progetto è stato versionato tramite Git e pubblicato su GitHub come supporto alla documentazione del project work, i file di configurazione presenti nella repository contengono valori di sviluppo o placeholder. Per l'utilizzo in ambienti reali è necessario impostare credenziali, chiavi e parametri di sicurezza appropriati.
