# Sistema di preventivazione sanitaria

Sistema di preventivazione sanitaria è una web application full-stack API-based progettata per supportare la gestione amministrativa e la preventivazione in ambito sanitario.

## Descrizione del progetto
Il progetto è stato sviluppato come applicazione web full-stack API-based per supportare la digitalizzazione dei processi amministrativi, gestionali ed economico-finanziari in un’organizzazione del settore sanitario.

## Tecnologie utilizzate
- ASP.NET Core (.NET 8)
- C#
- SQL Server
- Entity Framework Core
- ASP.NET Core Identity
- Bootstrap
- Swagger / OpenAPI

## Funzionalità principali
- gestione categorie professionali
- gestione coperture professionali
- gestione coperture accessorie
- gestione documenti
- creazione e consultazione preventivi
- autenticazione e gestione ruoli
- documentazione API tramite Swagger
